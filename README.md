# BossNotifier Fika Fix



**GitHub:** [kabzon93region](https://github.com/kabzon93region)

**Клиентский мод для SPT 4 + Fika.** Синхронизация каталога BossNotifier в headless-coop — уведомления о боссах работают на всех клиентах.



## Требования



- **BossNotifier** — оригинальный мод

- **Fika** headless-coop

- **SPT**: 4.0.x

- **BepInEx**: 5.4.x



## Установка



1. Скопировать BossNotifierFikaFix.dll в BepInEx/plugins/



## Совместимость



- headless_all — и на хосте, и на клиентах
